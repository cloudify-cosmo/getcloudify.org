h3. The Cloud Driver Functionality

The Cloud Driver is a layer used by the cloud controller to manage the infrastructure:

* Bootstrap Cloudify Management on the Cloud
<img src="/guide/2.1/images/overview/bootstrap_management.png"/>
* Provision and Terminate Cloud Resources for the Applications Services
<img src="/guide/2.1/images/overview/provision_app_vm.png"/>

The Cloud Driver has built-in implementations:

* Microsoft Azure
* OpenStack
* Citrix Cloud Stack (Previously Cloud.Com Product)
* EC2
* Rackspace
* Terramark
* Eucalyptus
* And more
