This topic provides instructions for bootstrapping Cloudify on RackSpace Cloud.

h2. Prerequisites

Before you bootstrap your RackSpace Cloud, make sure that your environment meets all the minimum requirements. For more information, see "Bootstrapping Prerequisites":../bootstrapping/bootstrapping_process.html#prereqs and "Configuring RackSpace":../setup/configuring_rsopenstack.html

h2. Bootstrapping

Bootstrapping RackSpace Cloud is extremely simple, just follow the "bootstrapping":../bootstrapping/bootstrapping_process.html#boot procedure, and type the following command in step 2: *@bootstrap-cloud@* @rsopenstack@

h2(#tear). Teardown

To shut down the RackSpace Cloud environment, at the Cloudify shell prompt, 
and type: *@teardown-cloud@* @rsopenstack@

h4(info). This uninstalls applications and shuts down all the management and service instance machines.
